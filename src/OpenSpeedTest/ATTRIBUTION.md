## Modifications

This fork has been customized for Ozark Connect / Network Optimizer:

1. **Branding** - Updated colors, logo, and styling to match Ozark Connect branding
2. **Geolocation** - Added optional location capture for network diagnostics
3. **Configuration** - Built-in configuration injection for result reporting
4. **Bug Fixes** - Fixed missing `OpenSpeedTestdb` variable
